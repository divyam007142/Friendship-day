## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/divyam007142/Friendship-day.git
cd Friendship-day
````

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```
